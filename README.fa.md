# 🔎 price-finder-bot | ربات قیمت‌یاب

**[English](README.md) · فارسی**

</div>

---

<div dir="rtl">

رباتی برای **تلگرام** و **بله** که نام یک کالا را از کاربر می‌گیرد، آن را در چندین فروشگاه اینترنتی ایرانی جستجو می‌کند، و نتیجه را به‌صورت یک **فایل اکسل مرتب** (از ارزان به گران، همراه با لینک خرید) به کاربر تحویل می‌دهد.

> کاربر می‌نویسد: «روغن» ← ربات جستجو می‌کند ← فایل اکسلِ مقایسه‌ی قیمت‌ها را دریافت می‌کند.

## ✨ امکانات

- پشتیبانی همزمان از **تلگرام** و **بله** با منطق مشترک
- جستجوی خودکار در فروشگاه‌های اینترنتی ایرانی با شبیه‌سازی رفتار انسانی (Playwright)
- خروجی **اکسل فارسی و راست‌به‌چپ** شامل:
  - خلاصه‌ی قیمت (کمترین، بیشترین، میانگین)
  - نام محصول، قیمت، نام فروشگاه، موجودی، امتیاز فروشنده، لینک خرید
  - مرتب‌شده از ارزان به گران
- سیستم **چند-ارائه‌دهنده‌ی هوش مصنوعی** با مسیر جایگزین (در صورت نیاز به پردازش زبان)
- ساختار ماژولار: هر فروشگاه یک ماژول جدا برای نگهداری آسان

│   └── ...                  # یک ماژول برای هر فروشگاه

│   ├── aggregator.py        # تجمیع و پاک‌سازی نتایج

## 🛒 فروشگاه‌های هدف

اولویت‌بندی بر اساس بیشترین پوشش و کمترین پیچیدگی:

| اولویت | فروشگاه | آدرس | توضیح |
|:---:|---|---|---|
| ۱ | ترب | torob.com | موتور مقایسه‌ی قیمت — صدها فروشگاه را یکجا پوشش می‌دهد |
| ۲ | دیجی‌کالا | digikala.com | فروشگاه عمومی بزرگ |
| ۳ | باسلام | basalam.com | بازار تولیدکننده‌ها |
| ۴ | عمده‌بازار | omde-bazar.ir | عمده‌فروشی |
| ۵ | اوکالا | okala.com | سوپرمارکت آنلاین (قیمت وابسته به موقعیت جغرافیایی) |
| ۶ | اسنپ‌شاپ | snappshop.ir | فروشگاه عمومی (اپ‌محور) |

علاوه بر این‌ها، فهرستی از فروشگاه‌های تأمین‌کننده‌ی اختصاصی نیز پشتیبانی می‌شود.

## ⚙️ نصب و راه‌اندازی

</div>

```bash
# کلون کردن مخزن
git clone https://github.com/USERNAME/price-finder-bot.git
cd price-finder-bot

# ساخت محیط مجازی
python -m venv venv
source venv/bin/activate      # در ویندوز: venv\Scripts\activate

# نصب وابستگی‌ها
pip install -r requirements.txt

# نصب مرورگر Playwright
playwright install chromium
```

## 🧰 ابزارها و فناوری‌ها

- **زبان:** Python
- **اسکرپینگ:** Playwright (مرورگر واقعی، شبیه‌سازی رفتار انسانی)
- **پیام‌رسان‌ها:** Telegram Bot API، Bale API
- **خروجی:** Excel با openpyxl (فارسی / راست‌به‌چپ)
- **پردازش زبان (اختیاری):** Groq، Google Gemini، OpenRouter

## ⚠️ نکات مهم

- بسیاری از فروشگاه‌ها سیستم **ضد ربات** دارند؛ اسکریپرها به نگهداری دوره‌ای نیاز دارند.
- اسکرپینگ با مرورگر **کند** است؛ توصیه می‌شود از **صف کار** برای جلوگیری از قفل‌شدن ربات استفاده شود.
- **موقعیت جغرافیایی سرور** بر دسترسی به فروشگاه‌های ایرانی و سرویس‌های هوش مصنوعی اثر مستقیم دارد و باید پیش از استقرار نهایی مشخص شود.
